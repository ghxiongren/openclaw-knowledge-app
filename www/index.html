📖什么是 OpenClaw？

OpenClaw 是一个强大的 AI 智能助手框架，让你可以：

  • 创建和管理多个 AI Agent（智能体）
  • 通过多种渠道与用户交互（Telegram、Discord、WhatsApp 等）
  • 使用丰富的插件系统扩展功能
  • 在本地或云端灵活部署

⚡核心特性

  • 多模型支持：支持 OpenAI、Anthropic、Google 等多种 AI 模型
  • 技能系统：通过 Skills 轻松扩展功能
  • 工作空间管理：独立的 workspace，持久化记忆
  • 异步任务：长时间任务后台执行，不阻塞对话
  • 安全优先：严格的权限控制和审计机制

🚀快速开始

安装

# 通过 npm 安装 npm install -g openclaw

初始化

# 创建新的工作空间 openclaw init my-workspace # 启动 Gateway 服务 openclaw gateway start

配置

编辑 ~/.openclaw/config.json 添加你的 API 密钥和偏好设置。

🧩技能系统

Skills 是 OpenClaw 的核心扩展机制，每个技能都是一个独立的功能模块。

内置技能

  • weather：获取天气信息
  • qqbot-cron：QQ 智能提醒任务
  • find-skills：发现和安装新技能
  • healthcheck：系统安全检查
  • agent-browser：浏览器自动化

安装技能

# 从 ClawHub 搜索技能 clawhub search weather # 安装技能 clawhub install weather

💬多渠道支持

OpenClaw 支持通过多种平台与用户交互：

  • Telegram：通过 BotFather 创建机器人并配置
  • Discord：支持服务器和私聊
  • WhatsApp：需要注册 Business API
  • Signal：端到端加密通信
  • iMessage：macOS 上直接集成
  • QQ Bot：国内最常用的即时通讯平台

🧠记忆与上下文

每个 Agent 都有独立的记忆系统：

  • MEMORY.md：长期记忆，存储重要决策和经验
  • memory/YYYY-MM-DD.md：每日日志，记录当日事件
  • SOUL.md：定义 Agent 的个性和行为准则
  • USER.md：记录用户偏好和上下文
💡 提示：每次对话开始时，Agent 会自动读取这些文件来恢复上下文和记忆。

🛠️常用命令

# 查看状态 openclaw status # Gateway 服务管理 openclaw gateway start openclaw gateway stop openclaw gateway restart openclaw gateway status # 列出会话 openclaw sessions list # 查看帮助 openclaw help

🔒安全与隐私

  • 数据隔离：每个 Agent 有独立的 workspace
  • 权限控制：严格的工具使用权限
  • 审计日志：所有操作都有完整记录
  • 本地优先：数据默认存储在本地，云端可选
⚠️ 重要：不要在 GROUP CHAT 中加载 MEMORY.md，避免个人隐私泄露。

  1. 善用 HEARTBEAT：定期检查邮件、日历等被动任务
  2. 记忆持久化：重要决策要写入文件，不要只依赖"脑力"
  3. 技能模块化：将复杂功能拆分成独立的 Skill
  4. 安全第一：执行外部操作前先确认
  5. 保持学习：定期更新技能和模型配置
