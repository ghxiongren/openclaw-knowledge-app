## 🚀 安装方法

### 方法 1: 直接安装（推荐）

1. **将 APK 传输到手机**
   - 通过 USB 数据线
   - 使用微信/QQ 等聊天工具发送
   - 使用网盘下载

2. **在手机上启用「未知来源」应用安装**
   - 设置 → 安全/隐私 → 允许安装未知来源应用
   - 或在安装 APK 时，系统会提示开启

3. **点击 APK 文件进行安装**
   - 系统会提示应用权限
   - 确认安装即可

### 方法 2: 使用 ADB 安装（需要 USB 调试）

```bash
# 1. 手机开启 USB 调试
# 设置 → 关于手机 → 连续点击版本号 7 次
# 返回设置 → 开发者选项 → USB 调试

# 2. 连接手机到电脑

# 3. 安装 APK
adb install OpenClaw-Knowledge-1.0.0.apk
```
